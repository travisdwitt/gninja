# GNinja
A shuriken slinging arcade game

<img width="580" height="414" alt="menu" src="https://raw.githubusercontent.com/travisdwitt/gninja/refs/heads/main/demo.gif" />

## Controls

- **Left/Right**: Move
- **Up**: Jump
- **Space**: Throw shuriken

## Installation
```bash
go build -o gninja
./gninja
```
